## Setup

```bash
# Clone the repo
git clone https://github.com/yourusername/monarx-Sentinel.git
cd monarx-Sentinel

# Create virtual environment
python3 -m venv .venv

# Activate virtual environment
source .venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Run
python main.py
```

## Requirements

- Python 3.8+
- Linux
